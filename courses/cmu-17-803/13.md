# 13

这份笔记（13-experiments-pt1.pdf）主要介绍了**实验设计（Designing Experiments）**的第一部分内容，重点是建立因果关系、不同类型的实验设计以及参与者的分配方式。

### 1. 实验的核心目标：确立因果关系

实验设计的主要目的是回答关于**因果关系**（Cause and effect）的问题，并确定**效果的大小**（Size of effect）。

**因果关系和反事实（Counterfactual）**

*   **效果**的定义是：将受试者接受某种处理（原因存在）时发生的事情，与如果同一批受试者**同时没有**接受该处理时可能发生的事情（即**反事实**）进行比较。
*   由于反事实无法被直接观察，因此必须进行**推断或近似**。
*   实验设计旨在**创建一种高质量但必然不完美的、用于推断反事实的来源**。

### 2. 实验研究的类型

笔记区分了多种研究类型，它们在对原因的控制程度上有所不同：

| 研究类型（Design Type） | 特点 | 目的 |
| :--- | :--- | :--- |
| **真实验（True Experiment）** | **设计了比较**，并对自变量（独立变量）的值进行**随机分配**（Random assignment）。 | 回答因果关系和效果大小的问题。 |
| **准实验（Quasi-Experiment）** | 原因通常无法被操纵。将自然发生的事件（如地震）与比较条件进行对比。 | |
| **自然实验（Natural Experiment）** | 又称**观察性研究（Observational study）**。仅观察变量之间关系的大小和方向。 | |
| **相关性研究（Correlational Study）** | 仅观察变量之间关系的规模和方向。 | |

### 3. 常见的实验设计模式（Schematics）

笔记使用标准符号（R=随机分配，O=观察/测量，X=干预/处理）展示了各种实验设计：

*   **基本 X vs C（处理 vs 控制）：** $R \ \ X \ \ O$ 对比 $R \ \ \ \ \ \ O$。
*   **基本 XA vs XB vs C（多处理 vs 控制）：** $R \ \ XA \ \ O$ 对比 $R \ \ XB \ \ O$ 对比 $R \ \ \ \ \ \ O$。
*   **前测-后测（Pretest-posttest）：** $R \ \ O \ \ X \ \ O$ 对比 $R \ \ O \ \ \ \ \ \ O$。使用前测（Pretest）可以增加对流失（attrition）的**稳健性**。
*   **交叉设计（Crossover）：** 受试者依次接受不同的处理，并在其间进行观察。
*   **阶乘设计（Factorial）：** 包含多个独立变量（因子），能更容易地测试处理组合，并允许检验**交互作用**（Interactions）。
*   **纵向研究（Longitudinal）：** 检验效果随时间的变化。

### 4. 参与者分配方式

研究人员通常根据参与者暴露于实验条件的方式，采用以下两种设计：

*   **组间设计（Between-Group / Between-Subject Design）：** 每位参与者**只暴露于一种**实验条件。优点是没有学习效应。例如，每个参与者只使用三种键盘中的一种来完成任务。
*   **组内设计（Within-Group / Within-Subject Design）：** 每位参与者**暴露于多种**实验条件。例如，每个参与者使用所有三种键盘来完成不同的文档。

**应对顺序效应（Order Effects）：**

*   在组内设计中，**学习效应**（Learning，有利于后完成的条件）和**疲劳效应**（Fatigue，对后完成的条件有负面影响）可能会带来混淆。
*   为了抵消顺序效应，可以使用**拉丁方（Latin Square）**设计来进行**平衡抵消（Counterbalancing）**，确保每种处理在序列中跟在其他每种处理之后的机会相等。

实验设计就像建造一座桥梁，**真实验**是通过随机分配和对照组来建造一座最坚固的桥（实现最清晰的因果推论），而**准实验**和**自然实验**则是在没有完全控制的情况下，试图从现实世界中已有的结构中推导出最可靠的连接。
