# 6

这份笔记（06 interviewing）详细介绍了定性研究中最常用的数据收集方法——**访谈（Interviews）**的方法论细节、实施步骤、目标和技巧。

### 访谈的性质和目标

访谈被认为是定性研究中最常见的数据收集方法。

*   **目标：** 访谈的目的是从受访者的角度看待研究主题，并理解他们**如何以及为何**形成这种特定的观点。
*   **研究类型：** 访谈是进行**探索性调查（exploratory investigation）**的绝佳工具，常用于**形成理论和假设**。访谈可以帮助收集历史数据（未被记录的数据），并以比书面交流更丰富的方式获取观点和印象。
*   **核心特征（定性访谈）：**
    *   访谈员施加的**结构化程度低**。
    *   主要使用**开放式问题**。
    *   重点关注受访者世界中的**具体情境和行动序列**，而非抽象和一般性意见。
    *   定性访谈将受访者视为**研究参与者（research participant）**，而不是被动回应问题的研究对象。

### 访谈的类型：半结构化访谈

在实践中，研究人员最常遇到或使用的是**半结构化访谈（Semi-Structured Interviews）**。

| 特征     | 特性（可能涵盖的范围）              |
| -------- | ----------------------------------- |
| **问题** | 从具体到一般                  |
| **顺序** | 从预先确定/固定到演变/灵活   |
| **回答** | 从精确到开放式                |

### 访谈的实施步骤

定性访谈通常包括以下五个主要步骤：

1.  **定义研究问题：** 重点是参与者如何描述和理解他们生活中的特定元素，**目的不是量化个体经验**。
2.  **创建访谈指南（协议）：** 访谈指南不是一套正式的、逐字逐句的提问清单。它是一个涵盖**主题（topics）**和用于跟进回答、引出更多细节的**追问（probes）**的列表。
3.  **招募参与者**。
4.  **执行访谈**。
5.  **分析数据**。

#### 访谈指南的创建

*   **结构：** 话题安排应从**开放式**转向**具体**，然后再回到开放式（例如：介绍 → 背景/历史 → 开放性问题 → 追问细节 → 其他要点 → 总结）。
*   **演变性：** 访谈指南可以在每次访谈后进行**修改**，包括添加在访谈中自然出现的主题或追问，并删除或重新表述那些令人费解或无法引出相关回应的问题。
*   **提问焦点：** 提问应侧重于**具体的例子**（concrete examples），而不是抽象的概括。

### 参与者招募和抽样

访谈通常采用**目的性抽样（Purposeful Sampling）**技术。

*   **抽样技术：** 包括典型案例、极端案例、关键案例、便利抽样和**最大差异抽样（Maximum Variation）**，其中最大差异抽样是首选，因为它为读者提供了最广泛的连接可能性。
*   **样本量：** 样本量的决定取决于**充足性（Sufficiency）**（反映人群中参与者和地点的范围）和**信息饱和度（Saturation）**（没有获得任何新的信息）。
*   **招募建议：** 最好通过**同伴**而非“上级”或“下级”进行接触。研究人员需要与受访者保持足够的距离，以免将某些信息视为理所当然。

### 访谈的执行技巧（Conducting the Interview）

访谈过程涉及激励、提问、倾听、理解、追问、保持控制、强化和记录八个方面。

#### 1. 激励与建立信任

*   **激励参与：** 访谈目的应与受访者的目标和价值观相关。
*   **营造氛围：** 创造一种让受访者感到被充分理解和安全的氛围，不会被评判或批评。
*   **回答疑虑：** 准备好回答关于研究目的、报酬、保密性、研究时长等问题。

#### 2. 提问和追问（Asking and Probing）

*   **提问原则：**
    *   **避免**同时问多个问题（例如：“你为什么加入这个开源项目，你认为它给你的编程经验带来了好处吗？”）。
    *   **避免**带有引导性（leading）的问题。
    *   使用**简单、直接、清晰**的方式提问。
    *   以容易回答且不会引起潜在尴尬或痛苦的问题开场（例如，请求事实或描述性信息）。
*   **开放式问题：** 提倡使用**开放式问题**，例如“全面描述式问题”（“Grand tour” questions），如“带我回顾一下你工作日的一天”。
*   **重建经验：** 要求参与者**重建（reconstruct）**他们的经验，而不是回忆（remember）。
*   **追问（Probes）：**
    *   **指令性追问（Directive Probes）：** 用于获取具体的步骤、细节，或从两极选项中进行选择。
    *   **非指令性追问（Non-Directive Probes）：** 例如，反映感受（“你觉得你没有得到公平对待？”）或间接跟进（“请多说一点”）。

#### 3. 倾听与控制（Listening and Controlling）

*   **倾听：** 关注言语和非言语线索，倾听完整的消息（事实和感受），并积极反馈以测试理解。
*   **少说多听：** 研究人员应少说话多倾听，可以通过检查录音中自己的段落长度来控制这一点。
*   **处理困难的受访者：** 对于言语过多的受访者，应在自然的停顿处礼貌地打断，并引导他们回到先前的话题。对于沉默寡言的受访者，应明确匿名性并使用沉默来鼓励回答。
*   **处理矛盾：** 应推迟追问矛盾，并在适当的停顿处以中立的方式反馈，帮助受访者解释看似冲突的观点。

#### 4. 记录和转录

*   访谈应**录音**，并**逐字逐句转录**。
*   在访谈过程中，仍应做**大量笔记**，因为查阅笔记比在录音中寻找特定内容更容易。

### 访谈的优缺点总结

| 优势 (Pros)         | 劣势 (Cons)                                     |
| ----------------------------- | ----------------------------------------------------- |
| 允许丰富的互动和后续追问。    | 样本量通常较小。                                      |
| 收集未被记录的历史数据。      | 耗时（包括访谈本身、转录和后续分析）。                |
| 详细地引出观点和印象。        | 寻找和安排合适的受访者具有挑战性。                    |
| 可以与其他数据源进行**三角验证**。 | 访谈者可能会引入潜在的**偏差**（bias），例如言语、语调或肢体语言可能影响回答。 |
