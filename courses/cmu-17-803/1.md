**二、研究方法论概述**

资料中介绍了两种主要的（但经常被错误混淆的）研究方法：

1.  **（客观主义）演绎研究方法 (The Objectivist Deductive Approach)**
    *   **别名：** 科学方法或实证科学。
    *   **假设：** (1) 存在一个独立于研究者的外部现实（真实世界）； (2) 可以通过收集关于该现实的客观、无偏见的数据来理解现实。
    *   **目标：** 通过对世界的因果机制建立日益完善的理解和洞察来积累知识。
    *   **性质：** 还原论（自上而下），从一般、抽象的概念化到特定情境中可观察、可衡量的数据。
    *   **典型流程：** 从抽象概念（理论）开始，推导假设，收集数据，测试假设。研究结果可能会证伪、支持、完善、挑战或扩展概念。
    *   **常用方法：** 实验（experiments）。

2.  **（主观主义）归纳研究方法 (The Subjectivist Inductive Approach)**
    *   **假设：** 现实是通过社会和经验构建的。
    *   **定性与定量方法的联系：**
        *   **定性（Qualitative）：** 适用于先验知识很少时，回答“发生了什么？”（“What’s going on?”）的问题。它提供**丰富（Rich）**的洞察。
        *   **定量（Quantitative）：** 适用于先验知识广泛时，回答“具体有何不同？”（“Precisely how is this different?”）的问题。它提供**精确（Precise）**的数据。
        *   **研究方法矩阵：** 定性方法包括人种学（Ethnography）、参与式观察、访谈等；定量方法包括真实实验、准实验、探索性数据分析、档案分析等。

**三、课程隐藏的主题与目标**

虽然这门课不是关于软件工程的（尽管会使用许多软件工程领域的阅读材料，但不需要软件工程的先验知识），但它实际上秘密关注以下主题：

*   **沟通 (Communication)：** 练习清晰地阐述问题、问题的重要性、解决问题的愿景和计划，以及传播研究结果（包括口头报告和期末研究报告）。有效的口头和书面沟通对于成功的科研是必需的，但比想象中要难得多。
*   **同行评审 (Peer Review)：** 学生将阅读并批判性地评审（包括书面形式）许多研究论文。
*   **培养健康的怀疑态度 (Skepticism)：** 学习实证研究方法可以帮助学生更好地识别任何科学论文的优点和缺点，从而“调整科学界的胡扯检测仪”（Tune your scientific BS meter!）。

**四、课程结构和评分**

*   **上课形式：** 分为讲座-研讨会模式。包括讲授新方法、课堂实践和学生展示示例论文。
*   **指导原则：** 优先考虑“像对待人类一样互相支持”（Prioritize Supporting Each Other as Humans）。
*   **评分构成：**
    *   **研究项目：** 50%（启动报告占 10%，最终报告占 30%，最终演示占 10%）。
    *   **其他作业：** 40%。
    *   **课堂演示与参与：** 10%（包括不定期的 Canvas 小测验）。
*   **项目/作业示例：** 包括进行文献综述、设计访谈方案或问卷、进行访谈并使用 LLM/ChatGPT 分析数据、应用特定的统计分析技术，以及批判一篇研究论文等。

**五、示例研究方法（01 Intro中提及）**

资料中提到了几种研究方法的实际应用示例，包括：

*   **人种学/访谈法：** 研究手术团队中的沟通和关系动态；研究面向对象语言中复制粘贴编程实践；通过深度访谈研究 GitHub 中的社交编码、透明度和协作。
*   **因果推断（如双重差分 DiD）：** 用于研究居家工作（WFH）对信息工作者协作活动的影响。
*   **随机试验：** **前瞻性随机社交媒体试验**，评估在 Twitter 上发布研究结果是否能提高论文引用量。

**形象化理解（定性与定量）**

定性研究（Qualitative）就像是**通过望远镜观察一个陌生的星系**（Prior knowledge: Very little, "What's going on?"），你获得了关于其形状和颜色的丰富（Rich）信息，这有助于你形成初步的理论。而定量研究（Quantitative）就像是**使用光谱仪分析你已知的恒星**（Prior knowledge: Extensive, "Precisely how is this different?"），你得到了关于其精确（Precise）的元素组成和运动速度，这使得你可以对现有理论进行严格的检验。两者从不同的角度揭示世界，且不应被错误地等同于归纳或演绎方法。
