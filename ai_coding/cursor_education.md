# Cursor Education

**Cursor** 是当前（2025年）非常热门的一款 AI IDE，内置强大的 Agent 能力，支持多种主流大模型（如 Claude），其操作体验与 VS Code 十分相似。事实上，笔者正在使用 Cursor IDE 编辑本篇文档。

不过，Cursor 的月费高达 20 美元，对于许多学生来说仍然是一笔不小的开销。好消息是，Cursor 提供了教育优惠：只需通过教育邮箱验证，即可免费使用一年。笔者本人就是借助在香港读书的同学的学校邮箱顺利完成了验证。不过，目前来看，Cursor 似乎对中国大陆的教育邮箱存在一定的地区限制，这部分还需要进一步调研和验证。
